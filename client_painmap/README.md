# Reddy_4_2_Front_End

# Installation & usage
## Instalation

1. Clone the repository
```sh
git clone https://github.com/Laplata247/reddy42-client.git
```  

2. Install required dependencies 
```sh
npm install
```

3. Run the client
```sh
npm run dev
```

The client should now be running on port 5173, or other port mentioned in the console.
